Mappa Mundi

Serious Game 2D

Présentation

The Albi’s Mappa Mundi is a serious game I developed during a 4-months internship at TECH’N’SMILE, a company specialised in Unity VR/AR application development. I was assigned to this project after the first 3D iteration was delivered. My job was to rework the whole application in 2D, propose new features and communicate with the client. The project was carried out following the SCRUM methodology.

Gameplay & Game Design

The application allows its users to visualize the world as it was perceived in the 8th century. Players can assemble a puzzle of the Mappa Mundi. Each piece gives them information about the selected region and its population habits in the middle age.

UX/UI

During the production, I worked with the UI-Designer on the application screens mockups. It allowed me to put in practice my ergonomy courses in a professional project and better understand the UX constraints in designing tactile applications.

Development

The game was developed using Unity. I developed the following features:

  • Tactile drag and drop puzzle pieces (translation and rotation)
  • Piece placement on the right puzzle location
  • Afk menu animation
  • Wrong piece placement animation (all animation were done with DOTween, a unity animation engine downloadable on the asset store)
  • Game tutorial and animations related to it
  • Scene transitions
  • Menu navigation and integration
  • Customisable information through an external excel file
  • 2D particle effects
  • multi-input management on tactile surfaces
