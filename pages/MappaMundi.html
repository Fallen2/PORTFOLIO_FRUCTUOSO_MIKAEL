Mappa Mundi

Serious Game 2D

Présentation

The Albi’s Mappa Mundi is a serious game I developed during a 4-months internship at TECH’N’SMILE, a company specialised in Unity VR/AR application development. I was assigned to this project after the first 3D iteration was delivered. My job was to rework the whole application in 2D, propose new features and communicate with the client. The project was carried out following the SCRUM methodology.

Unity 2D

Serious Puzzle Game

Gameplay & Game Design

  • Visualize the world as it was perceived in the 8th century
  • Assemble a puzzle of the Mappa Mundi and gather information about regions and population habits
  • Tutorial game design

UX/UI

  • Support on UX issues
  • UI integration
  • Feedback integration using the Unity animation engine DOTween

Development

The game was developed using Unity. I developed the following features:

  • Tactile puzzle pieces manipulation
  • Puzzle piece placement
  • Game tutorial
  • Scene transitions
  • Menu navigation and integration
  • Client-side administration through external config files
  • 2D particle effects
  • Multi-input management on tactile surfaces
